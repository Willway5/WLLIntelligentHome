## 在校项目
----
- 基于iOS手机客户端的智能家居远程控制
```
 负责服务端和手机客户端。服务端是在Centos 6.5 下采用LAMP(linux + apache + mysql + php)搭建的应用服务器。调用了Socket 实现和单片机的TCP连接,同时为手机客户端提供 post 和 get 接口。以服务器为媒介实现手机对家中各种电器的远程控制。
手机客户端 除了基础的UI 主要调用了AFNetworking 这个很流行的框架实现了与服务器的交 互。
```
